# office-hunting

# Get started

1. install xcodegen

`brew install xcodegen`

2. clone repo

`git clone https://github.com/Fuyan777/office-hunting.git`

3. move directory

`cd office-hunting`

4. move develop branch

`git checkout -b develop origin/develop`

5. create project using xcodegen

`xcodegen`

6. cocoapods setup

`pod install`
